# Ecommerce Store

This is a simple e-commerce store built with Next.js and Tailwind CSS. It uses
the following technologies:

- Next.js
- Tailwind CSS
- Prisma
- Stripe

## Features

- User authentication with Stripe
- Product listing and search
- Cart and checkout
- Payment processing with Stripe
- Admin dashboard for managing products and orders

## Technologies

- Next.js
- Tailwind CSS
- Prisma
- Stripe

## Deployment

The project is deployed on Vercel and can be accessed at [ecommerce-store.vercel.app](https://ecommerce-store.vercel.app).
